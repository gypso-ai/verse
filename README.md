## 📚 在线阅读器

仓库主页 [`index.html`](./index.html) 是一个零依赖的 Markdown 阅读器，
直接用来阅读上面两本小说。

特性：

- 一键切换 **短篇 / 长篇**
- 自动生成章节目录，点击跳转
- 三种主题：日 / 纸 / 夜
- 字号、行距可调
- 自动记忆上次阅读的书目和滚动进度（`localStorage`）
- 顶部阅读进度条 / 返回顶部按钮
- 移动端友好；纯 HTML + CSS + JS，单文件、无外部依赖

### 本地运行

由于阅读器通过 `fetch()` 加载 `.md` 文件，请使用本地静态服务器打开
（直接双击 `index.html` 在 `file://` 协议下浏览器会拒绝读取本地文件）：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000/
```

### 键盘快捷键

| 操作 | 按键 |
| --- | --- |
| 切换主题（日 → 纸 → 夜） | `T` |
| 关闭目录 / 设置面板 | `Esc` |
