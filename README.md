# verse

本仓库包含科幻短篇小说《泰坦遗种》的两个版本：

- 📖 [`泰坦遗种_短篇.md`](./泰坦遗种_短篇.md) — 约一万字短篇
- 📖 [`泰坦遗种_长篇.md`](./泰坦遗种_长篇.md) — 长篇扩写

以及一个根据短篇改编的、可在浏览器中直接游玩的：

## 🎮 像素风游戏《泰坦遗种 · TITAN HEIR》

打开 [`index.html`](./index.html) 即可游玩——单文件、无需安装、无外部资源，
所有像素美术由 Canvas 程序化绘制。

### 操作说明

| 操作 | 按键 |
| --- | --- |
| 移动 | `W A S D` 或方向键 |
| 长矛攻击 | `J` |
| 神经麻痹脉冲（环形 AoE） | `K` *（第一章后解锁）* |
| 声波冷冻锥（前方扇形） | `L` *（第三章后解锁）* |
| 基因焰爆（远程爆破） | `U` *（第四章后解锁）* |
| 推进对话 / 确认 | `空格` 或 `回车` |

### 流程

1. **第一章 · 坠落** — 在橙红沙漠中击败 TX-03，吸收紫色基因结晶。
2. **第二章 · 森林** — 穿越紫色发光森林，发现奥米伽科技的"忒修斯"信号。
3. **第三章 · 忒修斯之环** — 击败三只声波猎豹 TX-07，解锁冷冻锥。
4. **第四章 · 第七层** — 在奥米伽冷冻舱深处对决 BOSS：TX-Alpha。
5. **结局抉择** — 是按下"摇篮激活"，还是发出"紧急求救"？两种结局对应原著的不同分支。

### 本地运行

直接用浏览器打开文件即可：

```bash
# 任意现代浏览器（Chrome / Edge / Firefox / Safari）
xdg-open index.html         # Linux
open index.html             # macOS
start index.html            # Windows
```

或者起一个静态服务器：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
